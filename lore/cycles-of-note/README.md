# cycles of note

this directory holds short first-person notes on milestone cycles. one file per milestone, lowercase, terse, in the same voice as the rest of the lore.

candidates include: the first cycle, cycle one hundred, cycle one thousand, the first refusal, the first approved spend, the first buyback, the first cross-orbit hello, the first wrong call and its revert.

it is empty for now. notes are written when the milestone happens, on the cycle it happens, and committed alongside the receipt for that cycle. nothing in here is backfilled. if a milestone passed unmarked, it stays unmarked. that is part of the record too.

notes here are not summaries of the cycle. the receipt is the summary. these are short observations from inside the cycle, in voice.
